# trail_tp='L' 트레일링 스탑 최적화 보고서

## 1. 문제 분석

### 기존 로직의 한계
| 매도 조건 | 문제점 |
|-----------|--------|
| trade_tp='S': 최종이탈가(exit_price) 이탈 매도 | 고정 가격 → 주가 상승 후 되돌림 시 수익을 반납하고 손실 매도 |
| trade_tp='M': 이탈가(stop_price) 이탈 매도 | 동일 문제 |
| 15:10 전일저가 이탈 매도 | 15:10까지 대기하는 동안 손실 확대 |

**핵심 문제**: 이탈가가 매수 시점에 고정되어 주가 상승에 따라 올라가지 않음
- 예: 매수가 10,000원, 이탈가 9,500원 → 주가가 12,000원까지 상승 후 9,500원으로 하락 → -5% 손실 매도
- 동적 트레일이 있었다면: 12,000원 도달 시 스탑이 11,000원+ 로 상승 → +10%+ 수익 보전

## 2. 적용 전략: ATR + 수익구간별 하이브리드

### 2.1 ATR (Average True Range) 기반 동적 스탑

**ATR이란?**
- 14일간의 True Range(고가-저가, |고가-전일종가|, |저가-전일종가| 중 최대값) 평균
- 종목별 변동성을 자동으로 반영 → 변동성 높은 종목은 넓은 스탑, 낮은 종목은 좁은 스탑
- 데이터 부족 시 매수가의 3%를 기본값으로 사용

**계산 예시:**
```
종목 A (저변동성): ATR = 500원 → 스탑 거리 1,000원 (×2.0)
종목 B (고변동성): ATR = 2,000원 → 스탑 거리 4,000원 (×2.0)
```

### 2.2 수익 구간별 차등 적용

| 수익률 구간 | ATR 멀티플라이어 | 최소 보호 수익률 | 스탑 계산 |
|------------|-----------------|-----------------|----------|
| ≥ 10% (고수익) | ×1.5 (타이트) | 7% | `max(매수가×1.07, 최고종가 - ATR×1.5)` |
| ≥ 5% (중수익) | ×2.0 (중간) | 2% | `max(매수가×1.02, 최고종가 - ATR×2.0)` |
| ≥ 0% (소수익) | ×2.5 (넓음) | 원래 이탈가 | `max(이탈가, 최고종가 - ATR×2.5)` |
| < 0% (손실) | — | — | 원래 이탈가 (기존 로직 유지) |

### 2.3 동작 시나리오

**시나리오 1: 상승 후 하락 (수익 보전)**
```
매수가: 10,000원 | ATR: 500원 | 이탈가: 9,500원

09:30 종가 10,500원 (+5.0%) → effective_stop = max(10,200, 10,500-1,000) = 10,200원
10:00 종가 11,000원 (+10.0%) → effective_stop = max(10,700, 11,000-750) = 10,700원
10:30 종가 11,500원 (+15.0%) → effective_stop = max(10,700, 11,500-750) = 10,750원
11:00 종가 10,600원 (하락)   → 10,600 < 10,750 → 매도! (수익률: +6.0%)

기존 로직: 이탈가 9,500원까지 홀딩 → 최종 매도가 훨씬 낮을 수 있음
```

**시나리오 2: 하락 (기존 손절 유지)**
```
매수가: 10,000원 | ATR: 500원 | 이탈가: 9,500원

09:30 종가 9,800원 (-2.0%) → effective_stop = 9,500원 (원래 이탈가)
10:00 종가 9,400원 (-6.0%) → 9,400 < 9,500 → 매도! (수익률: -6.0%)

기존 로직과 동일: 손실 구간에서는 원래 이탈가로 방어
```

**시나리오 3: 횡보 (불필요한 매도 방지)**
```
매수가: 10,000원 | ATR: 500원 | 이탈가: 9,500원

09:30 종가 10,200원 (+2.0%) → effective_stop = max(9,500, 10,200-1,250) = 9,500원
10:00 종가 10,100원 (+1.0%) → effective_stop = max(9,500, 10,200-1,250) = 9,500원
10:30 종가 10,300원 (+3.0%) → effective_stop = max(9,500, 10,300-1,250) = 9,500원

소수익 구간에서는 ATR×2.5로 넓은 여유 → 정상 변동에 의한 조기 매도 방지
```

### 2.4 15:10 전일저가 이탈 조건 개선

| 구분 | 기존 | 개선 |
|------|------|------|
| 적용 대상 | 전체 (수익/손실 무관) | **손실 구간만** (gain_pct < 0) |
| 이유 | 수익 중인 종목도 전일저가에서 매도 | 수익 중이면 동적 트레일로 이미 보호됨 |

### 3.2 trail_tp='L' 블록 변경 (line 860~1022)

```
[기존]                          [변경 후]
                                ATR 초기화 (daily_data, atr_value, day_high_close)
                                    ↓
09:10 이전 스킵                  09:10 이전 스킵 (동일)
    ↓                               ↓
S: exit_price 이탈 → 매도       day_high_close 갱신
M: stop_price 이탈 → 매도           ↓
    ↓                           gain_pct 계산
15:10 전일저가 이탈 → 매도           ↓
                                수익구간별 effective_stop 계산
                                    ↓
                                close ≤ effective_stop → 매도
                                    ↓
                                15:10 전일저가 이탈 (손실 구간만)
```

## 5. 기대 효과

### 수익 극대화
- 주가 상승 시 `day_high_close` 추적 → 스탑이 자동으로 상승
- 고수익(10%+) 시 최소 7% 수익 보전 보장
- 중수익(5%+) 시 최소 2% 수익 보전 보장

### 손실 최소화
- 손실 구간에서는 기존 이탈가 유지 (악화 없음)
- ATR 기반이므로 저변동성 종목은 더 타이트한 스탑 자동 적용
- 15:10 전일저가 매도는 손실 종목에만 적용 → 수익 종목의 불필요한 매도 방지

### 변동성 적응
- 종목별 ATR 자동 계산 → 수동 이탈가 설정 의존도 감소
- 시장 변동성 변화에 자동 대응
